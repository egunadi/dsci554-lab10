Responsive D3 line/sparkline chart

The following implements a responsive line chart that changes to a sparkline when the width is less than 300px. The original code is from Observable Responsive Chart. The drawing code for the chart is in the JavaScript file charts/line_sparkline.js.

<script src="node_modules/d3/dist/d3.min.js"></script>
<script src="node_modules/d3-time-format/dist/d3-time-format.min.js"></script>

<svg id="chart" width="100" height="50"></svg>

<style>
  html,
  body {
    height: 100%;
  }
  svg {
    width: 100%;
    height: 100%;
  }
</style>

<link rel="stylesheet" href="charts/line_sparkline.css">
<script src="charts/line_sparkline.js"></script>
