In the following we are going to see how to create a responsive HTML chart by resizing text and marks based on the page width.

Responsive text

To adjust responsive text based on page width use 'vw' (viewport width) units. The viewport is basically the part of the document that is currently visible. 1vw is equal to 1% of the viewport width. In this case, the font size is set to 3% of the viewport width. This means that the text inside the <div> element will be 3% of the width of the viewport, regardless of the screen size or device used to view the webpage.

resizing the browser will resize this text to 3% of viewport width

<div style="font-size: 3vw;">resizing the browser will resize this text to 3% of viewport width</div>

Responsive div

To make div responsive we create the div inside a parent div and use percentage % units to specify the inner div width relative to the the parent div.

<div>
  <div style="display: inline-block; background-color:#7abcff; height: 40px; width: 80%;"></div>
</div>

<style>
  .title {
    margin-bottom: 10px;
    font-size: 2vw;
    /* responsive font size */
  }

  .item {
    /* wrapper for label + bar */
    text-align: left;
    margin-bottom: 5px;
  }

  .bar {
    display: inline-block;
    /* turn bar div inline so it shows on the same line with the label */
    text-align: right;
    vertical-align: middle;
    background-color: #7abcff;
    height: 30px;
  }

  .label {
    display: inline-block;
    /* turn label div inline so it shows on the same line with the bar */
    text-align: right;
    vertical-align: middle;
    line-height: 20px;
    /* make the text and bar the same height to they align */
    width: 20%;
    /* size of label relative to page */
    font-size: 2vw;
    /* responsive font size */
  }

  .value {
    /* the value label in the bar */
    color: #444;
    font-family: 'Courier New', Courier, monospace;
    /* use fixed font */
    font-weight: bolder;
    vertical-align: middle;
    /* align text to the middle of the bar */
    line-height: 20px;
    /* trick to make sure the label height is the same as the bar height so it will align to the middle */
    margin-right: 5px;
    /* move label 5px from the end of the bar */
    font-size: 2vw;
    /* responsive font size */
  }
</style>
